mdc -- markdown compiler (markdown-to-markdown)

Basic usage

mdc is an executable script (in Ruby), which read markdown from stdin and otuput compiled markdown to stdout.

mdc < in.md > out.md

or if you wish pandoc

mdc < in.md | pandoc -o out.html

@include (embedding)

mdc introduces new syntax, @[filetype](filepath). This leads

｀｀｀[filetype]
(content of filepath)
｀｀｀

e.g. @[ruby](example.rb)

dot (graphviz)

mdc finds dot code-blocks and translates to <svg> images.

Example

digraph {
    P -> {X Y};
    Z -> {X Y};
    Z -> P [style=dashed];
}

to a svg image (also see sample/out.html).
